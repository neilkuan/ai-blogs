三大使用情境冠軍

USE CASE / 01
多人使用
🥇 Google AI Pro

唯一支援家庭分享，最多 5 人各自獨立帳號 + 5TB 共享雲端。 其他兩家共用都違反條款。

USE CASE / 02
中年人 AI 入門
🥇 Google AI Pro

Gmail / Docs 原生整合零學習成本，NotebookLM Plus 文件問答超友善，5TB 存照片影片。

USE CASE / 03
CLI 額度爽度
🥇 ChatGPT Plus

同樣 $20，Codex CLI 給的訊息額度最多。 但 Claude Code 模型品質頂，看你重量還是重質。

— 老師的最終建議 —
若需求是「多人使用 + 給不會寫程式的 50 歲長輩入門」， 答案就是 Google AI Pro，毫無懸念。一份方案全家用，平均一人不到 $4/月。
  • 無痛融入既有 Gmail / Drive 使用習慣，不用學新介面
  • NotebookLM Plus：把保單、健檢報告丟進去用聊天問答
  • 5TB 雲端讓長輩存照片影片不再焦慮
  • 家庭分享，每人獨立帳號保有隱私
